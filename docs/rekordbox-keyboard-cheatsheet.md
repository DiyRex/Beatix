# KbdDJ 2-Deck — Rekordbox 7 Keyboard Cheat Sheet

**Left hand = Deck A · Right hand = Deck B**

## In the imported file (live now)

| Function              | Deck A (left) | Deck B (right) |
|-----------------------|:-------------:|:--------------:|
| Play / Pause          | Z             | /              |
| Cue                   | A             | '              |
| Beat Sync             | Q             | ]              |
| Hot Cue A / B / C     | 1  2  3       | 0  9  8        |
| 4-Beat Loop           | B             | N              |
| Loop Exit / Reloop    | G             | H              |
| Channel Fader Up/Dn   | C / X         | , / .          |
| Tempo Nudge Up/Dn     | Up / Down     | Right / Left   |
| Load track to deck    | Shift + Left  | Shift + Right  |

## FX — add in editor: click the (+) on the row, then press the key

| FX1 section row          | Key |
|--------------------------|:---:|
| FX1 On/Off               | 4   |
| Increase Parameter 1     | 5   |
| Decrease Parameter 1     | R   |
| (2nd FX) FX2/Parameter2 On/Off | 7 |
| (2nd FX) Increase Parameter 2  | 6 |
| (2nd FX) Decrease Parameter 2  | U |
| Select FX 1 / Assign Deck 1    | (pick effect + route to a deck) |

## EQ = MOUSE
Rekordbox does NOT allow EQ on the computer keyboard (Mixer only exposes
Trim / Gain / Fader / Headphones — no EQ). Drag the HIGH / MID / LOW knobs
with the mouse/trackpad.
Phrasing drill: with the mouse, cut LOW on the incoming track, then bring it
back up on the downbeat to blend.

## Notes
- Tempo nudge = momentary speed up/down to beat-match by ear (tap briefly).
- Channel fader is a step control (small/medium/large) — these keys use "large".
- "Trim value" IS keyboard-mappable if you ever want rough gain on keys.
